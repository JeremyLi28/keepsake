# Keepsake Pass

Give the ticket. Keep the memory.

Turn World Cup and event tickets into premium, collectible keepsake cards — personalize
the gift, then keep the memory forever.

## Development

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # typecheck + production build to dist/
npm run preview  # preview the production build
```

## Stack

- React 18 + TypeScript
- Vite

## Fonts

Newsreader (serif display) and Hanken Grotesk (body), loaded from Google Fonts.
